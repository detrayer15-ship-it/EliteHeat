# EliteHeat (Elite Education) — Образовательная платформа с ИИ-ассистентом

![EliteHeat Banner](https://img.shields.io/badge/Status-Hackathon--Finalist-brightgreen)
![Tech Stack](https://img.shields.io/badge/Stack-React%20%7C%20Node.js%20%7C%20Firebase-blue)

**EliteHeat** — это высокотехнологичная образовательная платформа нового поколения, созданная для обучения программированию (Python) и дизайну (Figma). Проект объединяет передовые веб-технологии, искусственный интеллект и глубокую геймификацию.

> [!IMPORTANT]
> Проект стал финалистом хакатона, набрав **40/38 баллов** на втором этапе и успешно пройдя защиту.

---

## 🚀 Ключевые особенности

- **🤖 ИИ-ассистент Mita**: Интеллектуальный ментор, доступный 24/7. Помогает студентам осваивать сложные темы Python и Figma простым языком.
- **🎮 Геймификация**: Система рангов (поясов), достижений и баллов, мотивирующая студентов к обучению.
- **💻 Интерактивное обучение**: Уроки с теорией, практикой и встроенной средой разработки (IDE).
- **🌐 3D-визуализация**: Использование **Three.js** для создания впечатляющих анимаций и интерактивных элементов (например, 3D-модель Земли).
- **💬 Real-time общение**: Интеграция чатов для живого взаимодействия.

---

## 🛠 Технологический стек

### Frontend
- **React + TypeScript**: Основа интерфейса.
- **Vite**: Быстрая сборка.
- **Tailwind CSS**: Современный адаптивный дизайн.
- **Three.js / Framer Motion**: 3D-эффекты и анимации.

### Backend
- **Node.js & Express**: Серверный API.
- **Google Gemini API**: «Мозг» ассистента Mita (модель 1.5 Flash).
- **Firebase**: Облачный хостинг, Firestore (БД) и аутентификация.
- **Socket.io**: Реализация чатов реального времени.

## 📦 Быстрый старт

### 1. Подготовка
```bash
git clone <url-вашего-репозитория>
cd EliteHeat
```

### 2. Установка зависимостей
```bash
# Фронтенд (из корня)
npm install

# Бэкенд
cd backend
npm install
```

### 3. Настройка окружения
Создайте файл `.env` в папке `backend`:
```env
GEMINI_API_KEY=ваш_ключ_gemini
PORT=3000
```

### 4. Запуск
```bash
# Бэкенд (из папки backend)
npm run dev

# Фронтенд (из корня проекта)
npm run dev
```
