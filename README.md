# EliteHeat — Образовательная платформа с ИИ-ассистентом

EliteHeat — это современная образовательная платформа, предназначенная для обучения программированию на **Python** и дизайну интерфейсов в **Figma**. Главной особенностью платформы является умный ассистент **Mita**, который помогает студентам осваивать сложные темы простым языком.

## 🚀 Особенности

- **ИИ-ассистент Mita**: Специализированный ментор по Python и Figma.
- **Интерактивное обучение**: Ответы на вопросы в реальном времени.
- **База знаний**: Более 40+ готовых ответов на типичные вопросы новичков.
- **Адаптивность**: Помощь на любом уровне — от основ до продвинутых тем.

## 🛠 Технологии

- **Frontend**: React, Vite, TypeScript, Vanilla CSS.
- **Backend**: Node.js, Express.
- **AI**: Google Gemini AI (модель 1.5 Flash).
- **Database**: Firebase (Firestore).
- **Deployment**: Firebase Hosting / Cloud Functions.

## 📦 Быстрый старт (Локально)

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url-вашего-репозитория>
   cd EliteHeat
   ```

2. **Установите зависимости**:
   ```bash
   # Фронтенд
   npm install
   
   # Бэкенд
   cd backend
   npm install
   ```

3. **Настройте переменные окружения**:
   Создайте файл `.env` в папке `backend`:
   ```env
   GEMINI_API_KEY=ваш_ключ_gemini
   PORT=3000
   ```

4. **Запустите проект**:
   ```bash
   # Бэкенд (из папки backend)
   npm run dev
   
   # Фронтенд (из корня проекта)
   npm run dev
   ```
