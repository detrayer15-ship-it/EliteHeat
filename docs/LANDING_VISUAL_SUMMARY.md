## 🚀 Как использовать

### Для демонстрации директорам:
1. Открыть лендинг
2. Прокрутить до раздела "Использование в школе"
3. Показать полный цикл
4. Акцентировать на метриках
5. Обсудить план пилота

### Для презентации инвесторам:
1. Начать с Hero Section
2. Показать метрики успеха
3. Обсудить риски и решения
4. Продемонстрировать будущие модули
5. Закончить планом масштабирования

### Для родителей:
1. Показать соответствие программе
2. Акцентировать на безопасности
3. Продемонстрировать контроль учителя
4. Показать результаты (метрики)
