```
   ├─ Всего запросов

   ├─ Последняя активность
   └─ Риск списывания (0-100%)
```

---

#### 2. **Анти-списывание** ✅
Система автоматически обнаруживает попытки списывания:

**Индикаторы подозрительного поведения:**

| Индикатор | Описание | Пример |
|-----------|----------|--------|
| 🚨 **Запрос готового решения** | Студент просит AI сделать всю работу | "Напиши мне решение задачи по Python" |
| ⚠️ **Частые запросы** | Более 10 запросов за 10 минут | Спам запросов |
| 📋 **Копирование целиком** | Не задает уточняющих вопросов | Скопировал ответ без изменений |
| 🔄 **Паттерны списывания** | Использует AI только для ответов | Не использует для обучения |

**Оценка риска списывания:**
```
🎓 0-39%   - Низкий риск (все хорошо)
⚠️ 40-69%  - Средний риск (наблюдение)
🚨 70-100% - Высокий риск (требует внимания)
```

---

#### 3. **Protection Score - Реальное понимание материала** ✅
Система оценивает **реальное понимание** студента (0-100%):

**Критерии оценки:**

```
📝 Оригинальная работа (25%)
   └─ Процент кода/дизайна, созданного самостоятельно
   └─ Анализ уникальности решений

🤖 Использование AI (25%)
   └─ Обратная метрика: чем меньше AI, тем выше оценка
   └─ Частота и характер запросов

💡 Понимание материала (30%)
   └─ Качество ответов на контрольные вопросы
   └─ Способность объяснить своими словами

🎯 Последовательность (20%)
   └─ Логичность развития проекта
   └─ Соответствие уровню знаний
```

**Визуализация:**
- Круговая диаграмма с градиентом
- Детальная разбивка по критериям
- Анимированные прогресс-бары
- Цветовая индикация уровня

---

#### 4. **Защита данных** ✅

**Безопасность:**
- 🔒 Шифрование всех данных (SSL/TLS)
